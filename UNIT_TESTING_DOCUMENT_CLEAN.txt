Action: User Authentication & Registration
Activities: Sign up new account, login/logout - enter credentials, authenticate, 
            create account with role
System Response: Correct login, secure session, account created successfully
Expected Error: Wrong credentials or duplicate email
System Response Error: Show error, do not log in or show "email already in use"
--------------------------------------------------------------------------------

Action: Booking Creation & Real-Time Availability
Activities: Create new booking - select dates, guests, pet option, view 
            availability calendar
System Response: Booking created as pending, calendar shows correct available 
                 and blocked dates
Expected Error: Past date, conflicting dates, or selecting unavailable date
System Response Error: Show validation error, date not selectable
--------------------------------------------------------------------------------

Action: Payment Proof Upload & Verification
Activities: User uploads payment proof (GCash/Maya/Bank), Staff/Admin verifies 
            and approves/rejects
System Response: File validated, payment status updated to verified/rejected
Expected Error: Invalid file, too large, or missing proof
System Response Error: Show upload validation error or notice
--------------------------------------------------------------------------------

Action: Booking Cancellation
Activities: Cancel existing booking - user or staff/admin triggers cancel with 
            reason
System Response: Status changes to cancelled, notification sent
Expected Error: Cancellation of already confirmed booking
System Response Error: Show cancellation policy message
--------------------------------------------------------------------------------

Action: Review Submission & Moderation
Activities: Submit review after completed stay (rate, write, add photos), system 
            auto-flags problematic content, staff/admin moderate flagged reviews
System Response: Review auto-published or flagged for review, approved reviews 
                 appear publicly
Expected Error: Review without required rating
System Response Error: Show validation error
--------------------------------------------------------------------------------

Action: Email & SMS Notifications
Activities: Booking events - create booking, confirm, cancel, verify payment, 
            send reminders
System Response: Correct emails and SMS sent with booking details
Expected Error: Email server problem or invalid phone number
System Response Error: Log error, show non-blocking message or phone validation error
--------------------------------------------------------------------------------

Action: Admin Dashboard & Reports
Activities: View dashboard - display statistics, bookings, payments, generate 
            reports, export to CSV
System Response: Dashboard shows accurate counts and charts, reports generated 
                 with correct data
Expected Error: No data available or invalid date range
System Response Error: Show "No bookings yet" or date validation error
--------------------------------------------------------------------------------

Action: Chatbot Support
Activities: Ask chatbot questions - type query, receive automated response
System Response: Chatbot provides relevant answers and guidance
Expected Error: Unrecognized question
System Response Error: Chatbot suggests alternatives
--------------------------------------------------------------------------------

Action: Profile Update & Account Settings
Activities: Update profile (name, phone), change password, export data 
            (JSON/CSV), delete account
System Response: Profile saved, password updated, data exported, account deleted 
                 if eligible
Expected Error: Invalid phone format, active bookings block deletion
System Response Error: Show validation error or blocking message with reason
--------------------------------------------------------------------------------

Action: Gallery Upload & User Role Management
Activities: Staff/Admin uploads resort photo, Admin changes user roles 
            (user/staff/admin)
System Response: Image uploaded successfully, role updated with new permissions
Expected Error: File too large, invalid format, or non-admin tries to change roles
System Response Error: Show upload validation error or access denied
--------------------------------------------------------------------------------

Action: Booking Features (E-Receipt & Reschedule)
Activities: User downloads e-receipt, reschedules booking with new dates
System Response: E-receipt generated and downloaded, booking dates updated with 
                 new price
Expected Error: Booking not confirmed or new dates unavailable
System Response Error: Show "receipt not available" or date validation error
--------------------------------------------------------------------------------

Action: Maintenance Mode
Activities: Admin activates maintenance - toggle setting
System Response: Booking disabled for users, maintenance message shown
Expected Error: Non-admin/non-staff tries to change
System Response Error: Access denied
--------------------------------------------------------------------------------

Action: Responsive Design
Activities: Test on desktop, tablet, mobile
System Response: Layout usable and functional on all devices
Expected Error: Very small screen
System Response Error: Layout adapts without breaking flows
